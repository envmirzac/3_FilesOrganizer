File Organizer

The File Organizer is a Python program that helps you organize files in a specified directory by moving them into appropriate subdirectories based on their file extensions.

Usage
Ensure that you have Python installed on your system.

Clone or download this repository to your local machine.

Open the file_organizer.py file in a text editor.

Modify the following variables in the code:

source_directory: Specify the path to the directory you want to organize.
destination_directory: Specify the path to the directory where you want to move the files based on their extensions.
Save the changes to the file_organizer.py file.

Open a terminal or command prompt and navigate to the directory where the file_organizer.py file is located.

Run the program by executing the following command:

python file_organizer.py
The program will organize the files in the source directory by moving them into subdirectories based on their file extensions in the destination directory.

After the program finishes execution, you will see a message indicating that the file organization is completed successfully.

Customization
You can modify the code to change the organization criteria, such as organizing files by file size, creation date, or any other file attribute.
